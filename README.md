# PYTHON_IA_CHATBOT
Uso de Python e IA para desenvolver uma ChatGPT a partir das práticas executados no curso da SoulCode.
Contém as seguintes etapas e comandos ao chatbot: 
### Preparar o ambiente;
###  Criando uma conta e obtendo a chave da API; 
### Chatbot V.1 - Estrutura e conversação;
###  Chatbot V.2 - Múltiplas mensagens;
### Chatbot V.3 - Contexto inicial;
### Chatbot V.4 - Controle de Temperatura;
### Chatbot V.5 - com Instruções: Inclui instruções adicionais ao chatbot;
###  Chatbot V.6 - Resposta não satisfatória;
###  Chatbot V.7 - Comandos personalizados;
###  Chatbot V.8 - integração com outra API e/ou serviço;
###  Chatbot V.9 - Armazerar e recuperar conversas anteriores.
