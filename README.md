# PYTHON_IA_CHATBOT
Uso de Python e IA para desenvolver uma ChatGPT a partir das práticas executados no curso da SoulCode.
Contém as seguintes etapas e comandos ao chatbot: 
### Preparar o ambiente;
###  Criando uma conta e obtendo a chave da API; 
### Chatbot V1 - Estrutura e conversação;
###  Chatbot V2 - Múltiplas mensagens;
### Chatbot V3 - Contexto inicial;
### Chatbot V4 - Controle de Temperatura;
### Chatbot V5 - com Instruções: Inclui instruções adicionais ao chatbot;
###  Chatbot V6 - Resposta não satisfatória;
###  Chatbot V7 - Comandos personalizados;
###  Chatbot V8 - integração com outra API e/ou serviço;
###  Chatbot V9 - Armazerar e recuperar conversas anteriores.
